# 🚀 NeoForge UI — Do Clone ao Produto Mínimo Viável (MVP+)

## 📌 Sobre o Projeto

O **NeoForge UI** é uma ferramenta web inspirada no conceito do Neumorphism.io, evoluída para um produto autoral com novas funcionalidades e identidade própria. O projeto foi desenvolvido como parte da atividade **“Do Clone ao Produto Mínimo Viável (MVP+)”**, utilizando engenharia reversa assistida por IA e integração com Firebase.

A proposta do sistema é permitir que usuários criem componentes neumórficos modernos de forma rápida, intuitiva e personalizada.

---

# ✨ Diferenciais Implementados

O projeto não se limita apenas à reprodução da ferramenta original. Foram adicionados novos recursos funcionais para transformar a aplicação em um produto mais completo.

## ✅ 1. Sistema de Login com Google

Integração com Firebase Authentication permitindo autenticação rápida e segura.

## ✅ 2. Salvamento de Presets

Usuários podem salvar suas configurações favoritas utilizando Cloud Firestore.

## ✅ 3. Histórico de Designs

O sistema armazena automaticamente os últimos designs criados pelo usuário.

## ✅ 4. Exportação de Componentes

Permite exportar estilos e componentes personalizados para reutilização em outros projetos.

# 🔥 Objetivo Acadêmico

O objetivo da atividade foi compreender como ferramentas de IA generativa podem auxiliar no desenvolvimento de software moderno, transformando o foco da programação manual para a modelagem lógica, análise funcional e refinamento de sistemas.

Além disso, o projeto explora os limites entre inspiração, engenharia reversa e inovação autoral no contexto atual do desenvolvimento assistido por IA.

# 📚 Reflexão

O desenvolvimento assistido por IA mostra como o papel do desenvolvedor está mudando. Hoje, além de programar, é necessário saber estruturar ideias, descrever sistemas com clareza e validar soluções geradas automaticamente.

Ferramentas como Gemini aceleram o processo de desenvolvimento, mas ainda exigem pensamento crítico, criatividade e conhecimento técnico para transformar protótipos em produtos reais.
